このシリーズの位置づけ

生成AIは、ただの「賢いチャットボット」ではありません。ユーザーが一文を送ると、その背後では、指示、会話履歴、ファイル、検索結果、ツール、メモリ、出力形式、実行権限が組み合わされ、ひとつの入力環境が作られます。

モデルはその中で次の応答やtool callを生成し、applicationはそれを検証し、runtimeは外部ツールを実行し、人間は目的・承認・責任を担います。

この超詳解版は、平易版より「わかりやすい版」ではありません。生成AIのしくみを、最も細かい責任境界と実行構造から読むための版です。

注記

このシリーズは、公開仕様、一般的なLLM / AI application設計、API設計上の概念、および理解のためのモデルを組み合わせた解説です。特定サービスの非公開内部実装をそのまま断定するものではありません。

prompt compilation という語は、非公開の内部処理名としてではなく、ユーザー入力、指示、履歴、メモリ、ファイル、tool定義などが、実行時に入力構造へ組み立てられる過程を理解するためのモデルとして扱います。

全体の読み方

01 request / prompt compilation
  ↓
02 instruction hierarchy
  ↓
03 context / memory / retrieval / files
  ↓
04 tool call / tool result / execution loop
  ↓
05 structured output / schema / validation
  ↓
06 memory / personalization / audit
  ↓

01〜05では、単発の応答がどのように組み立てられるかを見る。06では、過去情報がどのように保存・参照・注入されるかを見る。07では、agentがなぜ動き続けるのではなく、止まり、検証し、handoffする必要があるかを見る。08では、これらをmodel / application / runtime / humanの責任分担として統合する。

目次

01

プロンプトは実行時にどうコンパイルされるのか

ユーザーがチャット欄に書いた文章だけがpromptではない。system / developer instruction、履歴、memory、ファイル、検索結果、tool定義などが、実行時に入力環境として組み立てられる。

プロンプトとは、実行時に組み立てられた入力環境である。

02

指示階層とは何か

指示の強さは、言葉の強さではなく、置かれた層によって決まる。system、developer、user、tool resultを分け、prompt injectionを境界事故として理解する。

強い口調は、強い権限ではない。

03

contextとは何か

contextとは、保存情報そのものではなく、今回の生成に注入された入力集合である。履歴、memory、retrieval、file input、tool resultがどのように現在の応答に影響するかを見る。

contextとは、保存情報ではなく、今回の生成に入った情報である。

04

ツール実行ループとは何か

modelはtoolを直接実行しない。modelがtool callを出し、applicationが検証・実行し、tool resultをcontextへ戻す。

tool useとは、modelとruntimeのあいだのloopである。

05

構造化出力とは何か

構造化出力とは、きれいなJSONを書かせることではない。schema、parse、validate、refusal、retry、repair、handoffまで含む、applicationとの契約として扱う。

JSONが出たことと、扱える出力であることは違う。

06

memoryとは何か

memoryとは、モデル内部に人間の記憶のような体験が発生することではない。保存された情報が選別され、現在のcontextへ再注入される仕組みとして理解する。

memoryは、再注入されるcontext sourceである。

07

停止条件と検証ループ

agentic loopとは、モデルが自律的に動き続けることではない。停止条件、budget、verification、guardrails、handoff、traceによって制御されるloopである。

止まれることは、agentの能力である。

08

全体アーキテクチャ

生成AIシステムはmodelだけでは成立しない。model、application、runtime、memory、tools、guardrails、human、auditの責任境界を明示する。

AI system ≠ model alone.
