# genai-ron.jp 公開用統合パッケージ v5.7

生成日時：2026-05-20 JST

## 位置づけ

このZIPは、genai-ron.jp に論考②「理解・防御壁・行動」を追加するための公開用HTMLパッケージです。

## 含まれるもの

- トップページ更新案
- 論考一覧ページ更新案
- 論考②トップページ
- 論考② 第0章〜第12章ページ
- 論考② ビブリオグラフィーページ
- 論考②用の淡いグレージュ背景テーマ
- PDF導線の準備
- sitemap / robots / assets

## 推奨次工程

1. ブラウザで `index.html` を開いて導線・背景色・章ナビを確認
2. 問題なければ中間本文データから各章の全文を差し込み
3. PDF版作成
4. PDFリンクを有効化
5. sitemap.xml の公開日を確定日に更新
